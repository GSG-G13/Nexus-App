# Nexuus-App
Nexuus-App is a free photo sharing app. 

## Description 
Nexuus-App is a free photo sharing app. People can upload photos to our service and share them with their followers or with a select group of friends. They can also view like posts shared on Nexuus. Anyone can create an account by registering an email address and selecting a username.

---

## How to install?

- Clone the repo.
- npm install.
- npm run dev.
- Open http://localhost:4000/ in your browser.
- You can use the live demo of the project here.[Live Demo]('')
